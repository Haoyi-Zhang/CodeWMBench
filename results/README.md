# Results

This repository keeps only lightweight, repository-tracked summary outputs under `results/`.

- `schema.json`: schema notes for summary outputs
- `figures/dataset_statistics/`: repository-tracked dataset and evaluation-framework figures
- `tables/dataset_statistics/`: repository-tracked dataset statistics tables
- `figures/suite_all_models_methods/`: reserved export target for final full-run summary figures
- `tables/suite_all_models_methods/`: reserved export target for final full-run summary tables

Large raw per-run result trees are intentionally distributed outside git. See [`../docs/artifacts.md`](../docs/artifacts.md) and [`../ARTIFACTS.md`](../ARTIFACTS.md).
