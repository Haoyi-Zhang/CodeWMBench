# Figures

This directory contains repository-tracked summary figures that are safe to keep in the public repository.

- `dataset_statistics/`: tracked dataset and evaluation-dimensions figures
- `suite_all_models_methods/`: reserved export target for final full-run figures regenerated from released raw artifacts

In a fresh public clone, `suite_all_models_methods/` may contain only its README placeholder until raw results are restored or the compact full matrix is rerun.

The repository does not keep raw per-run plots or intermediate figure caches in git.
