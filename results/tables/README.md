# Tables

This directory contains repository-tracked summary tables that are safe to keep in the public repository.

- `dataset_statistics/`: tracked dataset inventory and compact-slice statistics
- `suite_all_models_methods/`: reserved export target for final full-run tables regenerated from released raw artifacts

In a fresh public clone, `suite_all_models_methods/` may contain only its README placeholder until raw results are restored or the compact full matrix is rerun.

The repository does not keep raw per-run reports or transient aggregation outputs in git.
