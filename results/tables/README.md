# Tables

This directory contains repository-tracked summary tables that are safe to keep in the public repository.

- `dataset_statistics/`: tracked dataset inventory and compact-slice statistics
- `suite_all_models_methods/`: materialized compact full-run summary tables shipped with the public companion repo

These table directories are the canonical home for exact-value leaderboard artifacts, benchmark-definition tables, run inventories, and timing summaries.

The repository does not keep raw per-run reports or transient aggregation outputs in git.
